# دليل لوحة الإدارة

افتح:

```txt
/#admin
```

## الأقسام

### نظرة عامة
يعرض ملخصًا سريعًا عن الإعدادات والروابط والأصوات.

### إعدادات الموقع
تعديل:
- اسم الموقع
- الدومين النصي الظاهر
- بريد الدعم
- البريد القانوني
- عنوان الصفحة الرئيسية
- وصف الصفحة الرئيسية
- حد أحرف الزائر
- حد أحرف المستخدم
- وضع الصيانة
- وضع التجربة

### السوشيال
إضافة وحذف روابط مواقع التواصل.

### الأصوات
إضافة وتعطيل الأصوات.

المهم هنا هو `model_voice_id`؛ هذا هو المعرف الذي سيرسل إلى خادم XTTS الخاص بك.

### المستخدمون
عرض المستخدمين وتغيير دورهم إلى Admin أو User، وحظر/فك حظر.

### التحويلات
يعرض آخر تحويلات TTS.

### القانوني
تعديل نصوص الشروط والخصوصية والكوكيز وDMCA وسياسة الأصوات.

### الدعم
عرض رسائل نموذج التواصل وإغلاق التذاكر.

### النظام
يعرض المتغيرات المطلوبة للتأكد من الإعداد.
