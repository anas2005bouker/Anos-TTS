# ملاحظات الأمان

1. لا تضع `SUPABASE_SERVICE_ROLE_KEY` داخل `src` أو داخل أي ملف Frontend.
2. مفاتيح `VITE_` فقط تظهر في المتصفح؛ لا تضع فيها أسرارًا.
3. إدارة المستخدمين تعمل عبر Netlify Function فقط.
4. RLS مفعّل على الجداول الأساسية.
5. ملفات الصوت في Supabase Storage خاصة، ويتم إنشاء signed URL مؤقت.
6. لا تفتح الاستخدام التجاري لـ XTTS-v2 قبل الحصول على ترخيص مكتوب.
