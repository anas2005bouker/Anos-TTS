# ترقية Anos TTS إلى v3

هذه النسخة تنظف الصفحة الرئيسية وتضيف إدارة محتوى الصفحة الرئيسية من لوحة الإدارة.

## الخطوات

1. ارفع محتويات هذا المشروع إلى GitHub فوق الملفات القديمة.
2. انتظر Netlify حتى ينتهي من النشر.
3. افتح Supabase > SQL Editor.
4. شغّل الملف:

```sql
supabase/upgrade_v3.sql
```

## ما الذي تغير؟

- الصفحة الرئيسية أصبحت موجهة للمستخدم فقط.
- أزيلت التفاصيل التقنية من الصفحات العامة.
- أضيف تبويب "الرئيسية" في لوحة الإدارة لإدارة نصوص الصفحة الرئيسية.
- روابط التواصل تظهر كأيقونات فقط للمستخدم، وتُعدّل كاملة من لوحة الإدارة.
- الخطة أصبحت مجانية + تبرع اختياري.
- الواجهة تعرض WAV فقط لأن الخادم الحالي يولد WAV.
- إدارة الأصوات توضّح أن كل صوت يحتاج ملف WAV فعليًا في Hugging Face.

## الأصوات المتعددة

لإضافة صوت جديد:

1. ارفع ملف WAV مرخص في Hugging Face داخل مجلد speakers.
2. من لوحة الإدارة > الأصوات، أضف صوتًا جديدًا.
3. في خانة "ملف WAV المرجعي" اكتب اسم الملف مثل:

```text
layan
```

أو:

```text
layan.wav
```

إذا لم يكن الملف موجودًا فعلًا في Hugging Face، لن يكون الصوت مختلفًا.
