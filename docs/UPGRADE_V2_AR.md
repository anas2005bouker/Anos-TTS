# تحديث النسخة الاحترافية v2

هذه النسخة تصلح المشاكل التي ظهرت في النسخة السابقة:

- تصميم أكثر احترافية وواجهة أنظف.
- إزالة النصوص التقنية التي كانت تظهر للمستخدم.
- إظهار روابط التواصل كأيقونات فقط.
- إضافة تعديل كامل لروابط التواصل من لوحة الإدارة، وليس حذف/إضافة فقط.
- جعل الإخراج WAV فقط لأن MP3 و OGG غير مفعّلين في خادم XTTS الحالي.
- صفحة حساب مستخدم رسمية أكثر: بيانات الحساب، الإحصائيات، وسجل التحويلات.
- صفحة الخطة أصبحت خيارين فقط: مجاني + تبرع اختياري 5 دولار قابل للتعديل.
- لوحة إدارة منفصلة لإعدادات الموقع، التبرع، الروابط، الأصوات، المستخدمين، التحويلات، الدعم، الصفحات، والنظام.
- صفحات: من نحن، الدعم، شروط الاستخدام، الخصوصية، إخلاء المسؤولية، سياسة الأصوات، الكوكيز، DMCA.

## 1) تحديث Supabase

إذا كنت نفذت `schema.sql` سابقًا، نفذ هذا الملف فقط في SQL Editor:

```text
supabase/upgrade_v2.sql
```

إذا كان المشروع جديدًا تمامًا، نفذ:

```text
supabase/schema.sql
```

## 2) متغير Netlify المهم

تأكد أن لديك هذا المتغير في Netlify:

```text
VITE_XTTS_API_URL=https://anas2005bouker-anostts.hf.space/tts
```

هذا يجعل الواجهة تتصل مباشرة بخادم XTTS، بعد أن ثبت أنه يعمل ويرجع `audio_base64`.

## 3) تحديث ملفات GitHub

ارفع محتويات هذه الحزمة إلى مستودع GitHub فوق الملفات القديمة، ثم انتظر Netlify حتى ينشر تلقائيًا.

الملفات الأهم التي تغيّرت:

```text
src/main.jsx
src/styles.css
package.json
supabase/schema.sql
supabase/upgrade_v2.sql
hf-space/app.py
hf-space/requirements.txt
hf-space/Dockerfile
```

## 4) تعدد الأصوات في XTTS

لا يوجد شيء اسمه "معرف صوت" جاهز يأتي من XTTS-v2 مثل ElevenLabs.

في XTTS-v2 الصوت يأتي من ملف WAV مرجعي. لذلك:

1. افتح Hugging Face Space.
2. اذهب إلى مجلد `speakers`.
3. ارفع ملفًا مرخصًا مثل:

```text
layan.wav
```

4. في لوحة الإدارة داخل الموقع، افتح تبويب الأصوات.
5. أضف صوتًا جديدًا:

```text
اسم الصوت: ليان
اللغة: ar
ملف الصوت المرجعي على الخادم: layan
```

يمكنك كتابة `layan` أو `layan.wav`.

الخادم سيبحث في:

```text
/app/speakers/layan.wav
```

إذا لم يجد الملف، سيستخدم `default.wav` كاحتياط.

## 5) ملاحظات مهمة

- لا تضف أصواتًا وهمية في لوحة الإدارة ما لم ترفع لها ملفات WAV فعلية.
- لا تعرض MP3 أو OGG إلا إذا عدّلت خادم XTTS ليحوّل WAV إلى هذه الصيغ.
- رابط PayPal أو أي منصة دفع يمكن تغييره من لوحة الإدارة > الدعم المالي.
